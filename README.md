## ðŸš€ Getting Started

### Prerequisites
- Flutter SDK 3.19 or higher
- Dart 3.0 or higher
- Android Studio/VSCode with Flutter extension

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/hanth0509/chat_ui_clone.git
cd chat_ui_clone
```

2. **Install dependencies**
```bash
flutter pub get
```

3. **Run the app**
```bash
flutter run
```

### For Development

4. **Run in debug mode**
```bash
flutter run --debug
```

5. **Build for production**
```bash
flutter build apk --release
# or
flutter build ios --release
```
